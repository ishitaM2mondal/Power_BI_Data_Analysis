# 📊 PhonePe Transaction Analytics Dashboard (Power BI)

## 📌 Project Overview

The **PhonePe Transaction Analytics Dashboard** is a Power BI project built using the **2024 PhonePe Transaction Dataset**. The dashboard provides a comprehensive analysis of digital payment transactions, including loans, bill payments, recharges, merchant payments, and other transaction categories.

The primary objective of this project is to transform raw transaction data into meaningful business insights through **data cleaning, data analysis, KPI generation, and interactive visualizations**. The dashboard enables users to monitor transaction trends, evaluate performance, compare monthly growth, and make informed business decisions with ease.
